☁️ 雲端同步模式（推薦）

家人在各自手機上都能看到同一份資料。需要先依照說明建立免費的 Supabase 資料庫（約 10 分鐘，僅需設定一次）。

還沒有 Supabase？看設定教學
  1. 到 supabase.com 註冊免費帳號並建立專案
  2. 在專案的 SQL Editor 貼上本專案提供的 setup.sql 並執行
  3. 到 Settings → API 複製 Project URL 與 anon public key
  4. 回到這裡貼上，自訂一組家庭代碼即可開始
